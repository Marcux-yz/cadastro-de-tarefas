# Cadastro de Tarefas (Task Registration)
Um sistema simples de cadastro de tarefas (A simple system for registering tasks)

## Como Funciona? (How does it work?)
1. O usuário cria tarefas (The user creates tasks)
2. O usuário edita tarefas (The user edits tasks)
3. O usuário apaga tarefas (The user deletes tasks)
4. O sistema persiste os dados (The system persists the data)

## Tecnologias Usadas (Technologies Used)
- UI: React
- Database: Local Storage

## Conceitos Aprendidos (Concepts Learned)
- Componentes (Components)
- Estados (States)
- Local Storage
- Condicionais (Conditions)
- Loops
- JSX/TSX
